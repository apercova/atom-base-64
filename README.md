# atom-base-64
Base64 encoder/decoder for Atom

### Export following comands for Base64 encoding/decoding:
- ctrl + e : Encode to Base64
- ctrl + d : Decode from Base64
- ctrl + o : Toggle Base64 Encoding/Decoding
